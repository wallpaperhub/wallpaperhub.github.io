How to Change the Font in Windows Terminal

To change the font in Windows Terminal, open the terminal and click on the dropdown arrow next to the tabs. Select "Settings" to open the configuration file. Under the "Profiles" section, choose the profile you want to edit. Look for the "fontFace" property and enter your desired font name. Save changes to see the new font.
