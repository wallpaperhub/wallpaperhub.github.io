Create & Subtract an Ellipse Shape: A Simple Guide

To create a new ellipse shape and subtract it from a background shape, start by drawing your desired background shape. Next, create an ellipse using the shape tool, positioning it where needed. Select both shapes and use the subtract option to remove the ellipse from the background, revealing the desired design. This technique enhances visual appeal!
